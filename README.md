# 🚀 Startup Sathi - Official Brand Website

The official landing page and client-generation platform for **Startup Sathi**, a creative marketing agency focused on helping businesses grow online with professional branding and digital marketing solutions.

This project is a lean, high-converting Single Page Application (SPA) designed for blazing-fast performance, premium UI/UX, and zero-maintenance backend lead capture.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Form Handling:** [Web3Forms](https://web3forms.com/)
* **Deployment:** [Vercel](https://vercel.com/)
* **Language:** TypeScript

## ✨ Key Features

* **Premium Animations:** Smooth scroll-reveals and staggered component loading using Framer Motion.
* **Responsive Design:** Fully mobile-optimized layout with a collapsing navigation bar and responsive pricing grid.
* **Serverless Lead Generation:** Contact form integrates directly with Web3Forms to route client inquiries straight to your email without needing a database.
* **Static Export Ready:** Built to be easily exported as pure HTML/CSS for ultimate speed and edge-network caching.
